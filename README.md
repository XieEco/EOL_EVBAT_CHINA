File 'GHGandCost.m' contains the main code for the disposal system.
File 'WithTransport.m' contains the main code for the results with inter-province transport.
File 'Without_transport.m' contains the main code for the results without inter-province transport.
All CSV files serve in 'Input data' folder are the input databases.
The 'MonteCarlo' folder contains the main code for MonteCarlo process.
The 'Uncertainty analysis' folder contains the main code for Uncertainty analysis process.  

The recommended environment is MATLAB R2016b.
